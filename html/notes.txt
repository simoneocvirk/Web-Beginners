HTML

- Attribute: src=""
	- 2 ways
		- Absolute URL - Ex: https://mcmaster.ca
		- Relative URL - Ex: sample.png
	- Example: <a href="sample.png">Visit McMaster</a>

- Attribute: width="" and height=""
	- <img src="sample.png" widtj="500" height="600">

- Attribute: alt=""
	- <img src="sample.png" alt="This is a sample image">

- Attribute: style=""
	- <p title="I'm a tooltip">This is a paragraph.</p>

CSS

- Cascading Style Sheets - to format the layout of HTML page

- 3 ways
	- Inline - by using the style attribute inside HTML elements
		- Ex: <p style="background-color: blue;"></p>
	- Internal - by using <style> element in the <head> section
		- <head>
			<style>
				body {background-cikir: powderblue;}
				h1 {color: blue;}
				p {color; red;}
			</style>
		  </head>
	- External - by using a <link> element to link to an external CSS file
		- Ex: <link rel="stylesheet" href="styles.css">
